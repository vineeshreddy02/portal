Basics



Open Notepad


START->Search->Notepad

Step 1:

Click on Search bar and type Notepad.

Step 2:

Select Notepad and Click on Open button.






Open an existing Notepad file


Step 1:

Press Ctrl + O on your keyboard or click Open from the File tab

Step 2:

Browse to your desired notepad file saved with .txt extension

Step 3:

Double click on that file to work on it.






Saving your notes


Step 1:

Press ctrl+Son your keyboard or Select Save from the File Tab

Step 2:

If you are saving your notes for the first time, then Save As dialog box will pop up and hence follow the following steps

Step 3:

In the File name , type a name for your notepad file.

Step 4:

Finally click Save to save the workbook completely.








Working in Notepad



Entering data

Any type of data â€“ numeric, alphanumeric, text or special symbols can be entered in the pad by typing.

To enter data just type the data that you wish to enter from the current cursor position.







Editing data

  1. Copy and paste

    2. To copy and paste your notes from one point to another follow these steps

    • Select the data using mouse or keyboard
    • Press Ctrl+C keys together on your keyboard or select copy from Edit menu
    • Press Ctrl+V keys together on your keyboard or select paste from Edit menu


  2. Clearing data partially or completely

    • Select the data that needs to be cleared
    • Press the delete key on your keyboard

  3. Finding data

    • Press Ctrl+F or select Find under Edit option from top menu
    • Enter the data to be searched
    • If data to be searched is present on left side of cursor select direction as Up else select Down
    • Click on find next


  4. Replacing data

    • Press Ctrl+H or select Replace under Edit option from top menu
    • Enter data which needs to be replaced in Find what box and the new data in Replace with box
    • Select replace or replace all as per your requirement


  5. Add current date and time to your sheet

    • Press F5 or select Time/Date under Edit option from top menu
    • Present date & time gets added to your current cursor position





Formatting data

Formatting text

  1. To change the font and Bold or Italic text

    1. Select the text to be formatted
    2. Click the Format tab => Font... The Font dialog box appears
    3. Click on various fonts to see a live preview of it.
    4. From the drop-down list of fonts, select the font that you want to use.
    5. Select the font style and press OK

  2. To change the font size

    1. Select the text to be formatted
    2. Click the Format tab => Font... The Font dialog box appears
    3. From the drop-down list of Size, select the font-size that you want to use and press OK.






Printing Sheet


First To setup the page, follow these steps:

  1. On the File menu, click Page Setup
  2. Choose the Size, orientation & margins as per your comfort
  3. Press OK and proceed for printing your sheet

To print the page, follow these steps

  1. On the File menu, click Print
  2. A print section will appear
  3. In this section, under General select the option you want.
    1. Select your printer
    2. Choose the number of copies that you desire
    3. Select the Paper type & Print Quality under Preferences tab
    4. Under Preferences => Paper/Quality, select the colour as Black & white or color as per your wish.
    5. Finally click on Print to confirm.
